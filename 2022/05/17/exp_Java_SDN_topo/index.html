思路

  • 由于Java调用北向接口过程复杂，故此处编写Python脚本获取拓扑
  • 使用Jython包完成Pyhon脚本的调用
  • 考虑到Python和Java的数据结构难以直接转换，这里选择将脚本获取的拓扑用无向图的形式保存本地文件topo.txt
