iptables note

发表于 2022-05-23 分类于 note

概念

作用

  • iptables 用于检查、修改、转发、重定向和丢弃 IP 数据包。

  • 过滤 IP 数据包的代码已经内置在内核中，并被组织成一组表，每个表都有特定的用途。

    • 这些表由一组预定义的链组成，链包含按顺序遍历的规则。

  • 每个规则由一个潜在匹配的谓词和一个对应的操作(称为目标)组成。

    • 如果谓词为真，则执行该动作；即条件匹配。

iptables允许用户使用这些链和规则
