分治

strassen算法

介绍

  • 用于方阵求乘积。

步骤

  • 将方阵分块后，得到8个子方阵；
  • 计算10个S矩阵；
  • 通过8个子方阵和10个S矩阵做7次矩阵乘法，得到7个P方阵；
  • 根据计算得到的P方阵组合求方阵乘积。

正确性证明

  • 将$P_1\cdots P_7$展开带入最后公式。

复杂度分析

  • 朴素算法求方阵乘积的时间复杂度是$O(n^3)$。
  • 方阵分块乘积时，若维度是2的倍数，可将每个方阵分为4个小方阵求乘积。
  • 得到时间复杂度公式如下

  • 推导出方阵分块求乘积时间复杂度为$O^3$。

  • strassen算法的时间复杂度在方阵分块的基础上将子方阵的乘法次数缩减为7。

  • 推导出strassen算法求乘积时间复杂度为$O^{lg7}$。

iptables note

发表于 2022-05-23 分类于 note

概念

作用

  • iptables 用于检查、修改、转发、重定向和丢弃 IP 数据包。

  • 过滤 IP 数据包的代码已经内置在内核中，并被组织成一组表，每个表都有特定的用途。

    • 这些表由一组预定义的链组成，链包含按顺序遍历的规则。

  • 每个规则由一个潜在匹配的谓词和一个对应的操作(称为目标)组成。

    • 如果谓词为真，则执行该动作；即条件匹配。

iptables允许用户使用这些链和规则
